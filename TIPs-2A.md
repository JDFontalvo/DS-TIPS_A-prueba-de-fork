### Suponga que tiene las tablas *SALAS* y *PELICULAS* compuestas de la siguiente forma.

### SALAS                     

| `PK` | Codigo | int |
|--------|--------|:--------:|
|  | Nombre | nvarchar(100) |
| `FK1`  | Pelicula  | int |
  

### PELICULAS
  
| `PK` | Codigo | int |
|--------|:--------|:--------:|
|  | Nombre | nvarchar(100) |
|  | CalificacionEdad  | int |
